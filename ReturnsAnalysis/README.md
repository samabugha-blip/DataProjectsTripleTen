## 2. Project Description

This project is an interactive Tableau dashboard designed to explore and analyze product returns using order and return data.  
The goal is to uncover *why* returns happen, *what groups drive the most returns*, and *what patterns emerge over time*.  
This helps businesses improve product quality, customer experience, and operational efficiency.
