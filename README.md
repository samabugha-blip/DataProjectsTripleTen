# DataProjectsTripleTen
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | NYC AirBnB Data | This spreadsheet project task was to analyze which neighborhoods and property sizes were most attractive for vacation rentals, and how much money these listings generate. |
| 2 | Returns Analysis | This Tableau project analyzes sales and profit performance across U.S regions, product sub-categories, and customer segments using the Superstore dataset provied by TripleTen. |
| 3 | Shopify App Analysis | In this PowerBI project I was hired to review the landscape of apps on the Shopify platform, using data scraped from publicly available Shopify websites. You want to figure out what key factors play into the success of a Shopify app. |
